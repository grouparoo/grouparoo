    • Defined in core/src/classes/plugin.ts:228

    Method to export many profiles to a destination Should only return a boolean indicating success, or throw an error if something went wrong. Can optionally return a processExports object to indicate exports will be processed asynchronously. Errors is an Array of Error objects with an additional profileId property so we can link the error to the specific export that caused the error. If there's a general error with the batch, just throw a single error.
